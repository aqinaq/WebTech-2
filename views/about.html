About CineShelf

What this project is about

CineShelf is a student project created to practice web development using HTML, CSS, and Node.js with Express

The goal is to browse movies, organize favorites, and explore movie details in a clean and simple interface

Planned Features

  • Search movies by title, genre, or year
  • Add movies to personal favorites
  • View movie details and ratings
  • Sort movies by release date or popularity
